ABSTRACT

Hand gesture recognition is an active research area in computer vision and machine learning, aiming to enable natural human-computer interaction without extra devices. The primary goal is to identify human gestures for conveying information or controlling devices. This paper presents a generic solution utilizing machine learning algorithms, achieving high accuracy in hand posture and dynamic gesture recognition. Applications include real-time systems for assisting soccer referees and interpreting Portuguese Sign Language, showcasing the versatility and potential of vision-based gesture recognition interfaces.

PAPER 1

Hand Gesture Recognition System based in Computer Vision and Machine Learning

Publication Year: 2019

Author: Paulo Trigueiros1,2,4, Fernando Ribeiro2,4 and Luís Paulo Reis

Journal Name: International Conference on Advances in Big Data

Summary: Hand gesture recognition is a natural way of human-computer interaction and an area of very active research in computer vision and machine learning. This is an area with many different possible applications, giving users a simpler and more natural way to communicate with robots/systems interfaces, without the need for extra devices. So, the primary goal of gesture recognition research applied to Human-Computer Interaction (HCI) is to create systems, which can identify specific human gestures and use them to convey information or controlling devices. For that, vision-based hand gesture interfaces require fast and extremely robust hand detection, and gesture recognition in real time. This paper presents a solution, generic enough, with the help of machine learning algorithms, allowing its application in a wide range of human-computer interfaces, for real-time gesture recognition. Experiments carried out showed that the system was able to achieve an accuracy of 99.4% in terms of hand posture recognition and an average accuracy of 93.72% in terms of dynamic gesture recognition. To validate the proposed framework, two applications were implemented. The first one is a real-time system able to help a robotic soccer referee judge a game in real time. The prototype combines a vision-based hand gesture recognition system with a formal language definition, the Referee CommLang, into what is called the Referee Command Language Interface System (ReCLIS). The second one is a real-time system able to interpret the Portuguese Sign Language. Sign languages are not standard and universal and the grammars differ from country to country. Although the implemented prototype was only trained to recognize the vowels, it is easily extended to recognize the rest of the alphabet, being a solid foundation for the development of any vision-based sign language recognition user interface system.

PAPER 2

IMU-Based Hand Gesture Interface Implementing a Sequence-Matching Algorithm for the Control of Assistive Technologies

Publication Year: 2021

Author: Frédéric Schweitzer 1,2 and Alexandre Campeau-Lecours ,2,*ORCID

Journal Name: IEEE conference paper

Summary: Assistive technologies (ATs) often have a high-dimensionality of possible movements (e.g., assistive robot with several degrees of freedom or a computer), but the users have to control them with low-dimensionality sensors and interfaces (e.g., switches). This paper presents the development of an open-source interface based on a sequence-matching algorithm for the control of ATs. Sequence matching allows the user to input several different commands with low-dimensionality sensors by not only recognizing their output, but also their sequential pattern through time, similarly to Morse code. In this paper, the algorithm is applied to the recognition of hand gestures, inputted using an inertial measurement unit worn by the user. An SVM-based algorithm, that is aimed to be robust, with small training sets (e.g., five examples per class) is developed to recognize gestures in real-time. Finally, the interface is applied to control a computer’s mouse and keyboard. The interface was compared against (and combined with) the head movement-based AssystMouse software. The hand gesture interface showed encouraging results for this application but could also be used with other body parts (e.g., head and feet) and could control various ATs (e.g., assistive robotic arm and prosthesis).

PAPER 3

Implementation and Analysis of AI-Based Gesticulation Control for Impaired People

Publication Year: 2019

Author: S. Nivash,1 E. N. Ganesh,2 T. Manikandan,3 Arvind Dhaka , 4 Amita Nandal , 4 Vinh Truong Hoang , 5 Adarsh Kumar , 6 and Assaye Bela

Journal Name: IEEE ACCESS

Summary: This study explores AI-based gesticulation control for impaired individuals. Previous methods using sensors and gloves for hand motion analysis posed challenges for elderly or those with chronic conditions. Lamberti and Camastra introduced a computer vision system with colored gloves, eliminating the need for sensors and addressing usability concerns. The experiment focused on hand signals and face motion, combining computer vision and face recognition for enhanced accessibility.
